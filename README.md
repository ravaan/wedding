# Wedding Invitation Website

A beautiful, mobile-first wedding invitation website for a Rajasthani-Sindhi wedding celebration in Nashik on April 24-25, 2026.

## Features
- Clean, minimal design with elegant animations
- Mobile-first responsive layout
- RSVP system with Google Sheets integration
- Event timeline and ceremony guide
- Travel and accommodation information
- Photo gallery
- Guest book for wishes and messages

## Technology Stack
- React 18+ with Vite
- Tailwind CSS for styling
- Framer Motion for animations
- React Hook Form for forms
- Lucide React for icons

## Development
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment
The website is configured for deployment on GitHub Pages initially, with plans to migrate to Netlify for custom domain support.
